The app is featured to do the basic todo operations, adding items to the list, editing the items within the list and deleting the ones completed.

## 1. Clone the Project
## 2. Install the dependencies
```
yarn
or
npm install
```
## 3. Run the app & start the bundle
```
yarn start
```
